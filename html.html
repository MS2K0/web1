HTML5

HTML je značkovací jazyk, který tvoří základ každé webové stránky. Bez něj by prohlížeč nevěděl, jak obsah zobrazit.

Co je HTML

HTML (HyperText Markup Language) je značkovací jazyk, který definuje strukturu webové stránky. Není to programovací jazyk - pouze říkáte prohlížeči, co je nadpis, co je odstavec, kde je odkaz a podobně.

Aktuální verze je HTML5, která přinesla nové sémantické značky jako <header>, <nav> nebo <article>. Díky nim je jasnější, co která část stránky představuje.

HTML Základní struktura 
<!DOCTYPE html>
<html lang="cs">
<head>
    <meta charset="UTF-8">
    <title>Můj web</title>
</head>
<body>
    <h1>Ahoj</h1>
</body>
</html>

Historie HTML

1991

HTML 1.0

Tim Berners-Lee vytvořil první verzi HTML s 18 značkami.

1995

HTML 2.0

První oficiální specifikace jako RFC dokument.

1999

HTML 4.01

Stabilní verze, která se používala řadu let.

2014

HTML5

Současný standard s novými značkami a podporou multimédií.

Sémantické značky

Sémantické značky mají význam - prohlížeč a vyhledávače díky nim vědí, co daná část stránky představuje. Je to důležité jak pro přístupnost, tak pro SEO.

<header>

Záhlaví stránky nebo sekce

<nav>

Navigační menu

<main>

Hlavní obsah stránky

<article>

Samostatný článek nebo příspěvek

<section>

Tematická sekce obsahu

<aside>

Vedlejší obsah, sidebar

<footer>

Zápatí stránky nebo sekce

<figure>

Obrázek s popiskem

Správný vs. špatný přístup

Starší přístup

<div id="header">
    <div id="nav">...</div>
</div>
<div id="main">
    <div class="article">
        ...
    </div>
</div>
<div id="footer">...</div>

Samé divy - prohlížeč netuší, co je co.

Moderní přístup

<header>
    <nav>...</nav>
</header>
<main>
    <article>
        ...
    </article>
</main>
<footer>...</footer>

Sémantické značky - jasná struktura.

Časté chyby

Chybějící alt u obrázků

Každý obrázek musí mít atribut alt s popisem. Bez něj nevidomí uživatelé nevědí, co obrázek zobrazuje.

❌ <img src="foto.jpg"> ✓ <img src="foto.jpg" alt="Západ slunce nad mořem">

Více H1 na stránce

Každá stránka by měla mít pouze jeden <h1>. Další nadpisy jsou H2, H3 a tak dále.

Neuzavřené značky

Každá otevírací značka potřebuje zavírací. Prohlížeč to někdy překousne, ale validátor na to upozorní.

❌ <p>Text bez konce ✓ <p>Text s koncem</p>

Tabulky pro layout

Tabulky jsou určené pro tabulární data, ne pro rozložení stránky. Pro layout se používá Flexbox nebo Grid.
