Validace kódu

Validace ověří, že je kód napsaný správně podle standardů. Validní kód funguje lépe a způsobuje méně problémů.

Co je validace

Validace je proces kontroly, zda kód odpovídá standardům HTML a CSS. Validní kód má několik výhod:

  • Kompatibilita - funguje ve všech prohlížečích
  • Přístupnost - asistivní technologie ho lépe čtou
  • Výkon - prohlížeč ho zpracuje efektivněji
  • Údržba - snazší hledání chyb

W3C Validátor

Hlavní nástroj pro validaci je validator.w3.org

Můžete validovat:

  • URL adresu webu
  • Nahraný soubor
  • Přímo vložený kód

Nejčastější chyby při validaci

Error Element not allowed as child

Značka je použita na místě, kde nesmí být (například div uvnitř odstavce).

Zkontrolujte vnořování elementů
Error Duplicate ID

Stejné ID je použité vícekrát. ID musí být v rámci stránky unikátní.

Použijte class místo ID, nebo ID přejmenujte
Error Missing required attribute

Chybí povinný atribut (nejčastěji alt u obrázků).

Doplňte chybějící atribut
Warning Empty heading

Nadpis neobsahuje žádný text.

Přidejte obsah nebo použijte jinou značku
Warning Section lacks heading

Element section by měl obsahovat nadpis (h2-h6).

Přidejte nadpis nebo použijte div
