Webové Standardy

Rozhodl jsem se zpracovat téma webových standardů, jelikož je to základ pro každého, kdo se chce věnovat vývoji webu. Projdeme si HTML5, CSS3, přístupnost a validaci.

Začít s HTML5

O čem to vlastně je

Webové standardy jsou pravidla a doporučení, jak psát HTML a CSS. Díky nim funguje web ve všech prohlížečích podobně a je přístupný pro různé uživatele. O standardy se stará organizace W3C, kterou založil Tim Berners-Lee.

Jelikož jsem se rozhodl zpracovat toto téma jako závěrečnou práci, tak jsem ho rozdělil do několika sekcí.

Témata, která zde probereme:

  • HTML5 - struktura a značky stránky
  • CSS3 - vizuální stránka webu
  • JavaScript - interaktivní prvky
  • WCAG - přístupnost pro hendikepované

Proč je to důležité

Kompatibilita

Web funguje v Chrome, Firefox, Safari i Edge. Nemusíte řešit, proč to někde nefunguje.

Přístupnost

Stránky mohou používat i lidé s postižením. Screen readery pak obsah přečtou správně.

SEO

Vyhledávače lépe pochopí strukturu webu a stránky se mohou lépe umístit ve výsledcích.

Výkon

Čistý kód prohlížeč zpracuje rychleji. To oceníte hlavně na mobilech.
