CSS3

CSS určuje, jak webová stránka vypadá. Zatímco HTML definuje strukturu, CSS se stará o barvy, velikosti a rozložení prvků.

Co je CSS

CSS (Cascading Style Sheets) je jazyk pro popis vzhledu webových stránek. Název "kaskádové" znamená, že se styly aplikují postupně a lze je přepisovat specifičtějšími pravidly.

CSS se většinou píše do samostatného souboru a propojuje se s HTML pomocí značky <link>. Díky tomu lze snadno změnit vzhled celého webu úpravou jednoho souboru.

CSS Základní syntaxe 
h1 {
    color: #264de4;
    font-size: 2rem;
    margin-bottom: 1rem;
}

Box Model

Každý HTML element je v CSS reprezentován jako obdélníková "krabice". Pochopení box modelu je klíčové pro správné rozvržení stránky.

margin
border
padding
content
  • Content - samotný obsah elementu
  • Padding - vnitřní odsazení
  • Border - ohraničení
  • Margin - vnější odsazení
* { box-sizing: border-box; }

Tento řádek je dobré mít na začátku každého CSS souboru.

Flexbox

Flexbox slouží k zarovnání prvků v jednom směru - buď v řádku, nebo ve sloupci. Je ideální pro navigace, karty nebo centrace obsahu.

CSS Flexbox 
.kontejner {
    display: flex;
    justify-content: space-between;
    gap: 1rem;
}

.polozka {
    flex: 1;
}

CSS Grid

Grid je určený pro složitější dvourozměrná rozložení. Hodí se pro celkový layout stránky nebo galerie obrázků.

CSS Grid 
.grid {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    gap: 1rem;
}

@media (max-width: 768px) {
    .grid {
        grid-template-columns: 1fr 1fr;
    }
}

Flexbox a Grid se dají kombinovat. Grid pro hlavní layout stránky, Flexbox pro menší komponenty uvnitř.

CSS Proměnné

CSS proměnné (custom properties) umožňují definovat hodnoty na jednom místě a používat je v celém souboru. Usnadňují změny designu.

CSS Proměnné v praxi 
:root {
    --barva-hlavni: #264de4;
    --barva-pozadi: #1a1a2e;
    --mezera: 1rem;
}

.tlacitko {
    background-color: var(--barva-hlavni);
    padding: var(--mezera);
}

Responzivní design

Web musí fungovat na všech zařízeních - od mobilů po velké monitory. K tomu slouží media queries.

CSS Media queries 
/* Základní styl - mobil */
.box { padding: 1rem; }

/* Tablet a výš */
@media (min-width: 768px) {
    .box { padding: 2rem; }
}

/* Desktop */
@media (min-width: 1024px) {
    .box { max-width: 1200px; }
}

/* Tisk */
@media print {
    .navigace { display: none; }
}

Běžně používané breakpointy

768px Tablety
1024px Desktop
1200px Velké monitory
